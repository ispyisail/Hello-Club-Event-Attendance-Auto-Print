Step 1: Configure Hello Club API Access

‚ÑπÔ∏è What you'll need: Your Hello Club API key from your admin account

To fetch event data, you need an API key:

  • Log in to your Hello Club admin account
  • Navigate to Settings ‚Üí API Keys
  • Create a new API key or copy your existing one
  • Paste it in the code block below
API_KEY=paste_your_api_key_here

Where to add this: Open the .env file in the project root folder and add the line above.

Step 2: Configure Email for PDF Delivery

‚ÑπÔ∏è Recommended: Use Gmail with an App Password for easiest setup

Choose how you want to deliver PDFs:

  • Email Mode (Recommended): Send PDFs to your printer's email address
  • Local Mode: Print directly to your default Windows printer

For Email Mode (Gmail):

  1. Enable 2-Factor Authentication on your Google account
  2. Go to Google App Passwords
  3. Create a new App Password for "Mail"
  4. Copy the generated password (spaces will be auto-removed)
PRINTER_EMAIL=your.printer@hp.com SMTP_USER=youremail@gmail.com SMTP_PASS=your_app_password_here SMTP_HOST=smtp.gmail.com SMTP_PORT=587

For Local Mode:

# In config.json, set: "printMode": "local"
